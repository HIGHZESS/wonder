마이페이지 - 지훈

마이페이지 첫화면 상단에 '아이디'님의 개인페이지입니다 노출됨

1. 일반회원 > 내정보 수정하는 기능 보여주기
1-1 일반회원으로 프리랜서 등록신청 정보적어서 제출하면 관리자에게 정보와함께 승인요청이 넘어가게되고 일반회원은 등급상태가 '승인대기'상태로 변경됨
1-2 신청을 안한상태로 [신청확인]버튼을 누르면 신청을 해야한다는 알림이 뜨고, 신청 후에 해당 버튼을 누르면  신청시 작성했던 정보 조회 및 현재 등급상태 확인가능


2. 프리랜서 > 내정보 수정할때 일반회원과 달리 추가로 더 많은 정보를 입력해야하고, 프로필사진 업로드시 미리보기가 가능함
2-1. 프리랜서 > 왼쪽 내 프로필사진 동그라미 누르면 프리랜서 상세보기 화면 나타남


4. 포트폴리오 관리에서 포트폴리오 사진 한번에 여러개 업로드하고 업로드된 사진을 클릭해서 모달창 보여주기(다시 큰 이미지 클릭하면 닫힘)


5. 찜해둔 상품 클릭시 상품목록에서 찜해뒀던 목록들 나오는데, 찜취소 버튼 눌러서 취소하는 기능도 확인


6. 거래내역 페이지에서 메세지 클릭하면 최초 메세지 등록되면서 메세지창으로 이동되는 기능 확인


7. 거래내역 페이지에서 의뢰서 클릭하면 모달창으로 의뢰서 뜨는것 확인 (x버튼 잘눌러야꺼짐 x버튼 근처 잘못누르면 안꺼짐 결국 수정못함 ㅠㅡㅠ)
7-1. 거래내역에는 상태가 총 5가지가 있다

의뢰대기 / 의뢰수락 / 결제완료 / 의뢰완료 / 의뢰취소



8. 결제버튼 누르면 api뜨는것 확인



9. 메세지 페이지에서 메세지 상대 선택하면 우측에 대화내용 불러오고 스크롤 자동으로 최하단으로 위치됨
메세지 상대를 선택하면 메시지 입력창 활성화되고 메세지 입력창에서 메세지 입력 후, 엔터키를 누르면 메세지가 전송되고(전송버튼 클릭도 가능)
쉬프트+엔터를 누르면 개행됨(두번누르면 안되더라구요 ㅋㅋ... 쉬프트엔터 한번만 해주세요)

개발자탭으로 창 하나 더켜서 상대방 아이디로 로그인한 다음, 메세지를 보내고
원래 내 아이디로 돌아와서 새로고침 버튼을 누르면 페이지 새로고침없이 ajax를 통해서 메세지가 불러와짐

메세지페이지의 모든내용은 ajax를 통해 제작됨



10. 암호변경메뉴에서 이전비밀번호 체크, 바꿀비밀번호 일치기능 ajax로 작동됨


11. 내정보 왼쪽 아레에 말풍선 모양 누르면 채팅페이지로 이동
12. 내정보 왼쪽 아래에 톱니바퀴 모양 누르면 내 정보페이지로 이동
13. 내정보 왼쪽 아래에 전원모양 누르면 로그아웃
